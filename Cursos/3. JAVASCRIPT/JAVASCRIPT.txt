# COMANDOS: typeof(variável): exibe o tipo de determinada variável.

variável: é um espaço na memória do computador destinado a um dado que é alterado durante a execução do algoritmo

var: escopo global e local, podendo ter seu valor alterado, e se não for declarado inicia-se como null;
let: escopo local; se não iniciada terá valor null.
const: escopo local, valor não alterado.

Escopo globoal: visível em todo o código;
escopo local: pode ficar disponível ou não;

comparação idêntica -> ===

### ARRAY ###

Array: arrays são um tipo de lista, ou matriz de variáveis, onde cada variável possui um índice. Os valores podem ser de vários tipos.

Declaração:
let array = ['string', 1, true, false, [‘array1], [‘array2’]...];

Acessando por índice:
console.log(array[0]);


### OBJETOS ### 

Dados que possuem propriedades e valores que definem suas características. Deve ser declarado entre chaves “{}”.

}
